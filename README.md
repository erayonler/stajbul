# StajBul

Bu web uygulaması, **2014–2025 yılları arasında** öğrencilerimizin staj yaptığı kurum ve kuruluşları harita ve liste görünümüyle sunmak amacıyla geliştirilmiştir.

## Özellikler
- Firma, şehir ve alan bazlı arama  
- Harita üzerinde konum gösterimi  
- İstatistiksel özet (firma, şehir, alan sayısı)  
- Stajyer arayan firmaların listelenmesi  

## Kullanılan Teknolojiler
- HTML, CSS, JavaScript  
- Leaflet.js (Harita)  
- MarkerCluster  

## Amaç
Öğrencilerin geçmiş staj deneyimlerinden faydalanarak kendilerine uygun staj yerlerini daha kolay bulmalarını sağlamak.
